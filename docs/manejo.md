# Manejo

<details>
<summary>Como transferir um aluno de uma turma para outra?</summary>

Acesse o menu **Alunos**, selecione o aluno, clique em **Editar** e ajuste os **vínculos de turma**.

</details>

<details>
<summary>Como consultar o histórico de movimentações do aluno?</summary>

No menu **Alunos**, selecione o aluno e acesse a seção **Histórico**.

</details>
