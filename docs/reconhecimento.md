# Reconhecimento

<details>
<summary>O aluno que não fizer o reconhecimento facial fica com falta?</summary>

Sim. O reconhecimento facial é o método oficial de registro de presença. Caso não seja realizado, o aluno será registrado como **ausente**.

</details>

<details>
<summary>Como faço para atualizar a foto do aluno?</summary>

No menu **Alunos**, selecione o aluno, clique em **Editar** e atualize a foto.

</details>
